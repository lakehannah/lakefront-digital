## Day 1: Welcome & Setup

### Morning (9:00 AM - 12:00 PM)

**Administrative (1 hour)**

- [ ] Set up email account: [name]@lakefrontdigital.com
- [ ] Add to company Slack/Discord

- [ ] Sign NDA and non-compete

**Orientation (2 hours)**
- [ ] Company overview: What we do, who we serve
- [ ] Walk through the contractor app prototype
- [ ] Review templates (general contractor, salon/beauty)
- [ ] Explain pricing: $2,500 / $5,000 / $1,500/month
- [ ] Show live site: https://lakehannah.github.io/lakefront-digital/
- [ ] Q&A session

### Afternoon (1:00 PM - 5:00 PM)

**Product Deep Dive (2 hours)**
- [ ] Demo the quote builder in contractor app
- [ ] Show how calculations work
- [ ] Explain LocalStorage vs backend
- [ ] Review existing templates
- [ ] Shadow Hannah on a mock sales call

**Tools Setup (2 hours)**
- [ ] Install CRM (HubSpot free tier or spreadsheet)
- [ ] Set up phone system (Google Voice)
- [ ] Connect email templates
- [ ] Create daily activity tracker

**End of Day 1:**
- [ ] Review what was learned
- [ ] Answer questions
- [ ] Assign Day 2 focus area

## Day 2: Market & Prospecting

### Morning

**Market Research (3 hours)**
- [ ] Drive around Spanish Fork/Provo area
- [ ] Identify 20 businesses without websites
- [ ] Take photos, write down names/addresses
- [ ] Categorize: contractors, salons, restaurants, etc.
- [ ] Input leads into CRM

**Lead List Review (1 hour)**
- [ ] Review existing lead list (spanish-fork-leads.md)
- [ ] Identify top 10 priority prospects
- [ ] Research each business online
- [ ] Prepare personalized pitches

### Afternoon

**Prospecting Practice (4 hours)**
- [ ] Write 3 cold email templates
- [ ] Write 3 cold text message scripts
- [ ] Practice cold call opening (role-play with Hannah)
- [ ] Send first 5 cold emails
- [ ] Make first 5 cold calls

**End of Day 2:**
- [ ] Review call/email results
- [ ] Adjust approach based on feedback
- [ ] Set activity goals for Week 1

---

## Day 3: Sales Process

### Morning

**Sales Process Training (3 hours)**
- [ ] Learn the full sales cycle:
  1. Prospecting
  2. Qualification
  3. Demo/presentation
  4. Proposal
  5. Close
  6. Handoff to production
- [ ] Review common objections and responses
- [ ] Practice handling "I need to think about it"
- [ ] Practice handling "You're too expensive"

**Demo Preparation (1 hour)**
- [ ] Create demo account in contractor app
- [ ] Add sample quotes
- [ ] Prepare sample customer data
- [ ] Practice walking through app features

### Afternoon

**Live Prospecting (4 hours)**
- [ ] Make 20 cold calls
- [ ] Send 10 cold texts
- [ ] Send 10 cold emails
- [ ] Log all activity in CRM
- [ ] Schedule 2-3 discovery calls

**End of Day 3:**
- [ ] Review day's activity
- [ ] Celebrate wins (even small ones)
- [ ] Plan tomorrow's focus

---

## Day 4: Closing & Contracts

### Morning

**Proposal Training (2 hours)**
- [ ] Learn to create proposals in contractor app
- [ ] Understand pricing breakdown
- [ ] Practice explaining value vs cost
- [ ] Review contract templates
- [ ] Learn deposit collection process

**Closing Techniques (2 hours)**
- [ ] Review assumptive close
- [ ] Review alternative close
- [ ] Review urgency close
- [ ] Practice closing on mock calls
- [ ] Learn when to walk away

### Afternoon

**Shadow Sales Calls (4 hours)**
- [ ] Listen to Hannah close a deal (if possible)
- [ ] Join as observer on prospect calls
- [ ] Take notes on what works
- [ ] Ask questions after each call
- [ ] Start running own calls with Hannah listening

**End of Day 4:**
- [ ] Debrief on shadow experience
- [ ] Identify personal strengths/gaps
- [ ] Set first week goals

---

## Day 5: First Week Review

### Morning

**Activity Review (2 hours)**
- [ ] Review all calls/emails from Week 1
- [ ] Analyze what worked and what didn't
- [ ] Adjust scripts based on real feedback
- [ ] Celebrate wins (calls made, meetings scheduled)

**Goal Setting (2 hours)**
- [ ] Set Week 2 activity goals (calls, emails, meetings)
- [ ] Set first month revenue goal
- [ ] Create daily schedule/routine
- [ ] Identify resources needed

### Afternoon

**Independent Work (4 hours)**
- [ ] Continue prospecting on own
- [ ] Run sales calls without shadowing
- [ ] Practice full sales cycle end-to-end
- [ ] Prepare for Week 2

**End of Week 1:**
- [ ] 1-on-1 with Hannah
- [ ] Review progress
- [ ] Adjust expectations if needed
- [ ] Plan Week 2 priorities
